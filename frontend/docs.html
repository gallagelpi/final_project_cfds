ðŸ“¡ How the API Works

The backend is built with FastAPI and serves a single endpoint:

POST /predict
  • Input: An image file (multipart/form-data).
  • Process:
    1. Receives the image bytes.
    2. Saves it temporarily to disk.
    3. Loads the trained ResNet18 model.
    4. Preprocesses the image (Resize to 224x224, Normalize).
    5. Runs inference on the GPU/CPU.
  • Output: JSON with the label ("alpaca" or "not_alpaca") and confidence score.

ðŸ“š Library Internals

The core logic resides in src/best_library, a custom Python package.

1. Preprocessing

We use standard ImageNet transformations:

  • Resize to 224x224
  • Convert to Tensor
  • Normalize (Mean: [0.485, ...], Std: [0.229, ...])

2. Model Architecture

We use Transfer Learning with ResNet18:

  • Backbone: Pre-trained on ImageNet (1.2M images).
  • Head: We replaced the final layer to output only 2 classes.

3. Training Strategy

The model was trained using:

  • Optimizer: Adam (Adaptive Learning Rate).
  • Loss Function: CrossEntropyLoss.
  • Hyperparameter Tuning: Grid Search over Learning Rate and Batch Size.
